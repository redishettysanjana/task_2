# 💰 Expense Tracker App

A simple yet effective **Expense Tracker Application** built using Python to help users record, manage, and review their daily expenses.

Developed as part of the **Maincrafts Internship**, this project focuses on strengthening core programming skills while solving a real-world problem—tracking personal finances.



## 📖 Overview

Managing daily expenses can get messy without a structured system. This application provides a lightweight, file-based solution using CSV storage to keep track of expenses efficiently.

It is designed to be:

* Beginner-friendly
* Easy to use
* Focused on core logic building

---

## ✨ Key Features

* ➕ Add new expenses with relevant details
* 📂 Store data persistently using a CSV file
* 📜 View all recorded expenses
* 🔍 Simple and readable data management
* ⚡ Lightweight and runs directly in the terminal

---

## 🛠️ Tech Stack

* **Language:** Python 
* **Storage:** CSV File (`expenses.csv`)
* **Core Concepts Used:**
  * File Handling
  * Functions
  * Conditional Logic
  * Data Processing
